### 📱 Développement Mobile

#### 🏀 [CoachFinder](https://github.com/Clayton2394/CoachFinder)
Une application mobile de mise en relation entre sportifs et coachs professionnels, intégrant un système de géolocalisation et de réservation en temps réel.
* **Technos :** React Native, Expo, Supabase (PostgreSQL, Auth).
* **Ce que j'ai fait :**
  * Création d'une interface UI/UX moderne avec React Navigation et flexbox.
  * Intégration de la géolocalisation pour afficher les coachs à proximité (`react-native-maps`).
  * Gestion sécurisée de l'authentification (rôles Joueur/Coach) et sécurisation des données avec les *Policies (RLS)* de Supabase.
